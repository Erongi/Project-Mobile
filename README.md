# React Reaction (RR)
### สร้างด้วย React-Native

โครงงานของเราเป็น แอพพลิเคชั่นเกมที่มีเป้าหมายฝึกให้ผู้ใช้งานมีปฏิกิริยาตอบสนองให้ว่องไวอีกทั้งยังเสริมปฏิกิริยาการแยกแยะให้ดีอีกด้วย
โดยโครงงานของเราได้ใช้ React Native ในการพัฒนาแอพพลิเคชั่นของเราเป็นหลัก
